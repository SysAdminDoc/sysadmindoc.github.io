// About

Who I Am

I'm a Senior Systems Administrator by day — provisioning workstations, managing networks, and keeping medical imaging systems running for healthcare facilities.

After hours, I build the tools I can't find anywhere else. What starts as a quick fix for a personal annoyance usually turns into a full-featured open source project. That pattern has produced 87+ public repositories across PowerShell, Python, JavaScript, Kotlin, and C#.

Everything I ship follows the same philosophy: zero configuration, dark theme, works on first launch. No setup wizards, no dependency hunts. If it takes more than one step to run, I haven't finished building it yet.

// Philosophy

What Drives Me

The principles behind every project I ship.

Zero Configuration

Download it, launch it, done. Every tool auto-installs its own dependencies, auto-configures prerequisites, and works on first run. No setup wizards, no README scavenger hunts.

Build What's Missing

When I can't find the right tool, I build it. What starts as a fix for a personal annoyance usually turns into a full-featured project that solves the problem for everyone.

Always Dark

Every interface ships with a premium dark theme. Deep palettes, glassmorphism, polished UI. I build tools I want to stare at for hours — and that means dark mode, always.

Open by Default

Everything I build is public. 90+ repositories and growing. I believe the best tools are free, transparent, and built to work for anyone in any environment.

// Journey

How I Got Here

Every project started with a problem nobody else was solving.

Healthcare IT & Systems Administration

Started in the deep end — provisioning workstations, managing networks, and keeping medical imaging systems running across healthcare facilities. Learned that the best way to survive complex environments is to automate everything you can.

PowerShell & Windows Automation

Built comprehensive Windows tools to solve real problems — NVMe driver patchers, network security auditors, Start Menu organizers, bulk installers. WPF interfaces that look and feel like premium software.

Python & Desktop Applications

Expanded into PyQt6 desktop apps — screenshot tools, firewall managers, image editors, PDF tools, and more. Every one bootstraps its own dependencies and launches with zero setup.

Web Apps & Live Deployments

Launched 12+ web applications on GitHub Pages — weather radar, flight tracking, OSINT aggregation, search engines. No backend required, no signup needed. Click a link and it just works.

Full Stack & Beyond

Grew to 90+ open source repos across five languages. Android apps in Kotlin, Chrome extensions, Tampermonkey userscripts, browser themes. If there's a platform, I've probably shipped something on it.
